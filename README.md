# 📧 SMS Spam Detection using Machine Learning

## 📌 Project Overview

This project implements a **Machine Learning based SMS Spam Detection system** that classifies messages as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) techniques.

Spam detection is an important application of machine learning used in email filtering, messaging systems, and cybersecurity.

---

## 🎯 Objective

The goal of this project is to build a model that can automatically identify whether a message is spam or legitimate based on its content.

---

## 📂 Dataset

The project uses the **SMS Spam Collection Dataset**, which contains labeled SMS messages categorized as:

* **Ham** – Normal messages
* **Spam** – Unwanted promotional or malicious messages

Dataset Size:

* **5572 SMS messages**

---

## 🧠 Machine Learning Techniques Used

### Text Preprocessing

* Lowercasing text
* Tokenization
* Stopword removal
* Removing punctuation

### Feature Extraction

* **TF-IDF Vectorization**

### Machine Learning Models

* **Naive Bayes**
* **Logistic Regression**
